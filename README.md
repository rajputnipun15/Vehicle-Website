# Vehicle-Website
A futuristic luxury autonomous AI vehicle website showcasing modern UI/UX design. Built using pure HTML, CSS, and vanilla JavaScript, it features a dark premium theme, glassmorphism effects, 3D hover animations, parallax scrolling, smooth transitions, and a fully responsive layout to demonstrate creative front-end skills.

Luxury Futuristic Autonomous AI Vehicle Website

A modern, visually rich front-end website showcasing a futuristic autonomous AI-powered vehicle. This project focuses on premium UI/UX design, smooth animations, and interactive effects using only core web technologies.

Features
- Dark luxury theme with gold and blue accents
- Glassmorphism UI components
- 3D hover effects and mouse-based tilt
- Smooth scrolling and parallax animations
- Cinematic hero section
- Fully responsive layout
- Clean and minimal design

Technologies Used
- HTML5
- CSS3 (3D Transforms, Animations, Glassmorphism)
- Vanilla JavaScript

Project Objective
This project was created to demonstrate front-end development skills, creative UI design, and interactive web effects without using any external libraries or frameworks. It is suitable for B.Tech academic submissions and portfolio projects.
